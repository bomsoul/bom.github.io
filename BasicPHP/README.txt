เปิดผ่านlocalhost
โปรแกรมที่ผมใช้ในเป็นLocalhost คือ Laragonนะครับ ติดตั้งโดยการเข้าไปที่ C:/laragon/www แล้วโยน
folderลงไปใน www หลังจากนั้นเปิด Laragon App แล้วกดStart all ตามด้วยเปิดBrowser แล้วพิมว่า
localhost/ชื่อโฟลเดอร์ หรือชื่แไฟล์ได้เลย

เปิดผ่านheroku 
uploadไฟล์ลงไปในgithubก่อน หลังจากนั้นให้เข้าไปที่herokuแล้วlogin ตามด้วยcreate new App ใส่ชื่อApp
หลังจากนั้นตรงdeploy ให้เลือกลิ้งเป็นgithubก่อน แล้วใส่ใส่ชื่อrepo github ที่ต้องการให้เชื่อม หลังจากนั้นกดdeploy
